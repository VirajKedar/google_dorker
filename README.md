# Google Dork Launcher  
### Launch Google Dorks directly from your Linux terminal.

![License](https://img.shields.io/badge/License-MIT-green.svg)
![Status](https://img.shields.io/badge/Status-Stable-brightgreen)
![Shell](https://img.shields.io/badge/Shell-Bash-blue)

Google Dork Launcher is a lightweight Bash tool that allows penetration testers, bug bounty hunters, and researchers to run Google Dork queries directly from their terminal.

It automatically URL-encodes your query and opens the Google results in your default browser.
